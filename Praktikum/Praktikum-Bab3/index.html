A. Pengenalan Form dalam HTML

Form digunakan untuk mengumpulkan data dari pengguna. Atribut 'action' menentukan tujuan pengiriman data, dan 'method' menentukan cara pengiriman (GET atau POST).

B. Input Text & Textarea

Input teks digunakan untuk masukan teks pendek seperti nama pengguna.




Textarea digunakan untuk masukan teks yang lebih panjang seperti komentar atau deskripsi.


C. Dropdown (Combobox)

Dropdown digunakan saat pengguna harus memilih satu opsi dari beberapa pilihan.


D. Radio Button & Checkbox

Radio button digunakan untuk memilih satu opsi dari beberapa pilihan yang ada. Semua input harus memiliki 'name' yang sama.




Checkbox digunakan untuk memilih lebih dari satu opsi.


E. Button & File Upload

Tombol digunakan untuk mengirim formulir atau menjalankan skrip.



File upload memungkinkan pengguna mengunggah file. Form harus memiliki atribut enctype="multipart/form-data".




F. Validasi Form Dasar

HTML menyediakan atribut untuk validasi dasar guna memastikan data sesuai format yang diharapkan sebelum dikirim.
